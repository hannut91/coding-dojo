# 두 정수 사이의 합

## 1. 문제

### 원하는 것

두 정수 a, b가 주어졌을 때 a, b사이에 속한 모든 정수의 합을 리턴하는 함수를 만들어라.

### 조건

a, b가 같은 경우는 a를 반환하라
a와 b는 -10,000,000 이상 10,000,000 이하인 정수입니다.
a와 b의 대소관계는 정해져있지 않습니다.

## 2. 계획

만약 a == b 이면 a를 리턴한다.
더 큰 값을 선택해서 같아질 때까지 반복문을 사용해 모두 더하고 반환한다.
