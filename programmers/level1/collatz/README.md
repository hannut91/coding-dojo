# 콜라츠 추측

## 1. 이해

### 원하는 것

주어진 숫자를 몇번의 콜라츠 작업을 해야 1이 되는지를 구하는 함수를 작성하여라

### 조건

500번을 넘는다면 -1을 반환한다.
입력된 수, num은 1 이상 8000000 미만인 정수입니다.

## 2. 계획

만약 입력된 수가 짝수라면 2를 나눈다.
아니면 3을 곱하고 1을 더한다.

만약 나온 결과가 1이라면 지금까지 몇 번 수행 했는지 결과를 출력한다.
아니면 위에 했던 일을 반복하고 카운트를 올린다.
