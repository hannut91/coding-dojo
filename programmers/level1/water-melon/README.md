# 수박수박수박수박수박수?

## 1. 문제

### 원하는 것

길이가 n이고, 수박수박수박수....와 같은 패턴을 유지하는 문자열을 리턴하는 함수를 만들어라.

### 조건

n은 길이 10,000이하인 자연수입니다.

## 2. 계획

n / 2 만큼 수박을 만들고 홀수면 마지막 한 개를 자르면 될 것 같다.
