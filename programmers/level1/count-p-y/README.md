# 문자열 내 p와 y의 개수

## 1. 문제

### 원하는 것

대문자와 소문자가 섞여있는 문자열 s가 있다.
s에 'p'의 개수와 'y'의 개수를 비교해 같으면 True, 다르면 False를 return 하는 solution를 완성하라.

### 조건

대소문자를 가리지 않는다.
하나도 없는 경우 True를 리턴.

## 2. 계획

regular expression을 사용해서 p의 개수와 y의 개수를 찾는다.
비교해서 리턴한다.
